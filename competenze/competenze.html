EPICODE FullStack Web DeveloperAprile 2023 / Novembre 2023

Svolgimento: Part-time serale dal Lunedì al Venerdì

Il programma di formazione è suddiviso in 7 moduli focalizzati su aspetti chiave dello sviluppo web con esami scritti e progetti finali a conclusione di ogni modulo. Durante il corso, ho affrontato sia la teoria che la pratica, lavorando su progetti di gruppo con l'uso di strumenti collaborativi come GitHub. I moduli:

  1. HTML e CSS: Fondamenti di sviluppo web, apprendendo come creare pagine web e gestire lo stile.
  2. JavaScript: Approfondimento nel linguaggio di scripting per implementare funzionalità dinamiche nelle applicazioni.
  3. Bootstrap e Responsive Web Design: Sviluppo di interfacce responsive, garantendo un'esperienza utente ottimale su diverse piattaforme.
  4. JavaScript Avanzato: Approfondimento nelle tecniche avanzate di JavaScript per gestire progetti più complessi.
  5. React: Esplorazione del framework React per la creazione di interfacce utente reattive e scalabili.
  6. Backend: Introduzione al lato server dello sviluppo web, imparando come creare API e gestire server e database.
  7. Capstone Project: Progetto finale in cui gli studenti mettono in pratica tutte le competenze acquisite.
